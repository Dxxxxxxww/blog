# css 记录

## 单/多行省略号

```css
@mixin nowrap($line: 1) {
    overflow: hidden;
    text-overflow: ellipsis;
    @if $line == 1 {
        white-space: nowrap;
    } @else {
        display: -webkit-box;
        -webkit-line-clamp: $line;
        /* autoprefixer: ignore next */
        -webkit-box-orient: vertical;
    }
}
```
