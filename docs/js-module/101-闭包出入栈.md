# 闭包出入栈

```js
function sayHi() {
  debugger
  const hi = 'hi'
  return function son() {
    debugger
    console.log(hi)
  }
}

const gSon = sayHi()

gSon() // gSon执行时，父层函数已经执行完出栈了。gSon 作用域中会保留着闭包中的变量。

// gSon 的作用域以及优先顺序：
// Local
//    this: window
// Closure (sayHi)
//    hi: "hi"
// Script
// Global -> window
```
