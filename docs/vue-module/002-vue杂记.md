## vue 数组为什么不进行 Object.defineProperty

尤大回答： 出于性能考虑，不对数组做 Object.defineProperty

```js
const arr = ['a', 'b', 'c']

let val = arr[0]
Object.defineProperty(arr, '0', {
  get: function () {
    console.log(123)
    return val
  },
  set: function (newVal) {
    val = newVal + 1
  }
})

console.log(arr[0])
arr[0] = 100
console.log(arr[0])
```

## vue 切换只有运行时版本和带编译版本代码

1. 使用别名

```js
// vue.config.js
module.exports = {
  // ...
  resolve: {
    alias: {
      vue$: 'vue/dist/vue.esm.js'
    }
  }
}
```

2. 使用 vue-cli 配置

```js
// vue.config.js
module.exports = {
  runtimeCompiler: true // 带编译版本
}
```
