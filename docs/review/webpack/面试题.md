## loader plugin 的区别

loader 用来帮助 webpack 识别它所不支持的文件。loader 用来加载资源文件，并可以对这些资源文件进行处理，最终输出到指定的文件中。是转译模块源代码的转换规则，资源文件加载器。

plugin 可以侵入 webpack 打包过程，可以理解为 webpack 打包编译时的生命周期钩子函数。用来做一些增强功能。
